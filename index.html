FAQ

Alles Wichtige rund um Installation und Keys.

Wie installiere ich HugoMacro?

Fabric Loader für 1.21.11 + Fabric API installieren, dann die .jar aus dem Dashboard in deinen mods-Ordner legen. Spiel starten – der Key wird automatisch geladen.

Welche Minecraft-Version wird unterstützt?

Minecraft 1.21.11 mit Fabric Loader und Fabric API. Java 21 oder neuer erforderlich.

Wie öffne ich das Dashboard im Spiel?

Standardmäßig mit der Taste P. Alle Tasten lassen sich in den Minecraft-Steuerungseinstellungen anpassen.

Wie bekomme ich einen Key?

Öffne ein Ticket im Discord und wähle deinen Plan. Den Key trägst du auf der Website ein, um die Mod herunterzuladen.

Kann ich auf mehreren Geräten nutzen?

Standard ist 1 Gerät. Ein zweites Gerät kann per Anfrage freigeschaltet werden (dann nutzen 2 Personen denselben Key).

Was passiert wenn mein Key abläuft?

Nach Ablauf der Laufzeit wird der Client automatisch gesperrt und die Macros lassen sich nicht mehr starten.
