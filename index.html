❓ Häufige Fragen

Alles Wichtige rund um HugoMacro.
Wie installiere ich HugoMacro?

Fabric Loader 1.21.11 + Fabric API installieren, .jar in den mods-Ordner legen, Spiel starten, Key eingeben.

Welche Minecraft-Version wird unterstützt?

Minecraft 1.21.11 (Fabric), Java 21+.

Wie öffne ich das Dashboard im Spiel?

Mit der Taste P. Anpassbar in den Minecraft-Steuerungen.

Kann ich auf mehreren Geräten nutzen?

Je nach Plan 1–2 Geräte. HWID wird automatisch beim Start gebunden.

Mein Key funktioniert nicht – was tun?

Öffne ein Ticket im Discord, wir helfen dir weiter.

Bekomme ich Updates?

Ja, alle Updates sind je nach Plan inklusive.
